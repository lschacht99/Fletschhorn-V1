# Waldhotel Fletschhorn / Private Alpine Chalet Saas-Fee

A complete vanilla HTML/CSS/JavaScript and Squarespace-ready website for repositioning Waldhotel Fletschhorn as a private alpine chalet / private hotel buyout in Saas-Fee.

Core promise: **a private mountain world in Saas-Fee, reserved for your people**. The site is for whole-property inquiries for families, groups, weddings, retreats, ski weeks and private events — not room-by-room hotel booking.

## Local preview

```bash
cd fletschhorn-private-estate
npm run preview
```

Visit `http://localhost:4173`, or open `preview/index.html` directly.

## Squarespace install

Use `squarespace/custom-css.css`, `squarespace/global-header-injection.html`, `squarespace/global-footer-injection.html`, and the seven code blocks in `squarespace/pages/`.

## Pages

- Home
- The Property
- Suites & Sleeping
- Weddings & Private Events
- Retreats & Groups
- Dining, Wine & Hospitality
- Saas-Fee & Inquiry

API proxy examples remain in `api/` for future Guesty/Getty or other property data normalization. Do not expose credentials in frontend JavaScript.
