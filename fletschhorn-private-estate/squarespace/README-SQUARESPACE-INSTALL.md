# Squarespace install

1. Paste `squarespace/custom-css.css` into Squarespace Custom CSS.
2. Paste `squarespace/global-header-injection.html` into Header Code Injection.
3. Paste `squarespace/global-footer-injection.html` into Footer Code Injection.
4. Create these seven pages in Squarespace: Home, Property, Suites & Sleeping, Weddings & Events, Retreats & Groups, Dining, Saas-Fee & Inquiry.
5. Paste the matching file from `squarespace/pages/` into each page Code Block.
6. Keep the conversion flow inquiry-first; do not connect instant room booking widgets.
7. Deploy an API proxy later only when credentials are available.
